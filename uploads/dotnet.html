dotnet - .NET Command Line

dotnet commandDescription
dotnet new listList all templates.
dotnet new [template] [name] [--output output_folder]Create a new project using a template. (e.g., dotnet new webapi --name MyApi)
dotnet add package Add references to external libraries. (e.g., dotnet add package Microsoft.EntityFrameworkCore.SqlServer)
dotnet add [PROJECT] reference [PROJECT_REFERENCES]Enable code sharing between the projects. (e.g., dotnet add MainProject.csproj reference LibraryProject/LibraryProject.csproj)
dotnet watchAutomate development by restarting/reloading your application on code changes.
dotnet buildBuild source code into an executable or library.
dotnet runExecute the built application directly from the development environment.
dotnet publishOptimize and packages the application for deployment to a production environment.
dotnetRun unit tests in the project.
dotnet restoreRestore NuGet packages specified in the project file.
dotnet --helpDisplay general help information.
dotnet [command] --helpProvide help specific to a command. (e.g., dotnet build --help)
